HTML QUESTIONS

  1. What is html
    • HTML stands for HyperText Markup Language.
    • It is the standard language used to create and design web pages.
  2. What is the difference between HTML and XHTML
    • HTML
      • HyperText Markup Language.
      • Based on SGML (Standard Generalized Markup Language)
    • XHTML
      • eXtensible HyperText Markup Language
      • Based on XML (eXtensible Markup Language)

  4. What is Element
    An HTML element is the fundamental building block of an HTML document, representing a specific type of content or structure on a webpage, such as a paragraph, heading, image, or link
  5. What is the difference between HTML4 and HTML5
    • HTML4
      • Released in 1997
      • Long doctype declaration.
      • Needs plugins (like Flash) for audio and video
    • HTML5
      • Finalized in 2014
      • Short doctype: !DOCTYPE html5
      • Includes semantic tags like header, footer, section
      • Built-in support for audio and video
  6. What is list
    A list in HTML is a structured way of displaying related items, either in a specific order or without order. It uses list tags to group items together, making them easier to read and understand
    1. Ordered list
    2. Un-Ordered list
    3. Description list
